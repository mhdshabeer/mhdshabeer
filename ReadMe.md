## 🚀 Featured Project

### Trading Buddy AI

An event-driven AI-powered trading journal assistant designed to automate trade journaling and performance analysis.

#### Workflow

Trade Closes  
→ MT5 Detection  
→ Telegram Notification  
→ Voice Note Input  
→ Whisper Transcription  
→ LLM Psychology Extraction  
→ PostgreSQL Storage  
→ Notion Journal Creation  
→ Natural Language Analytics  

#### Key Features

- Automated trade journaling
- Voice-to-text trade reviews
- Psychology and bias extraction using LLMs
- Real-time Telegram interaction
- PostgreSQL analytics database
- Notion integration
- Market news and economic event digests
- Natural language querying of trading history

#### Tech Stack

Python • PostgreSQL • Telegram Bot API • LLMs • Whisper • Notion API • MT5 Integration • Async Workflows
